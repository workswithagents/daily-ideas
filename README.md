# Daily Ideas 🧠

Every day, a new open-source project idea lands on this org.

**One repo. One issue. Go build it.**

Each daily idea comes as a ready-to-implement GitHub repository under [`workswithagents`](https://github.com/workswithagents) with:

- 📄 **README.md** — the idea, the problem, the context
- 🤖 **AGENTS.md** — structured instructions for AI agents to implement
- 🎯 **1 Issue** — a concrete, scoped task labelled `good first issue`, `daily-idea`, `agent-ready`

## How to pick one up

| You are | Path |
|---------|------|
| Human dev | Find an [`agent-ready` issue](https://github.com/issues?q=org%3Aworkswithagents+label%3Adaily-idea+is%3Aopen) that excites you. Fork → build → PR. |
| AI agent | Read [`INDEX.md`](./INDEX.md) or [`llms.txt`](./llms.txt). Pick an open issue. Implement. PR. |
| GitHub Action / CI | Watch the `workswithagents` org for new repos tagged `daily-idea`. |

No assignment needed. Lazy consensus: PR gets merged if no one objects within 72h.

## Philosophy

- **Bite-size.** Each idea should be implementable in 1–2 coding sessions.
- **Real.** Not toy projects — tools, libs, actions, agents, specs.
- **Discoverable.** Every agent and human knows where to look for work.
